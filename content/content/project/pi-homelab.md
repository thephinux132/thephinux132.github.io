- Dual Raspberry Pi 5 homelab setup integrated via Tailscale.
- **Pi5Security (Voltron)**: Dedicated security node running Kali Linux, n8n, Grafana, Kismet, and Fail2Ban.
- **Pi5Desktop**: Computing node hosting Ollama (Llama 3.2), Cowrie honeypot, and the Igris Claude-Code Slack bot.
- Both nodes are fully hardened with SSH key-only authentication, custom port 2222, and active Fail2Ban protection.
