Student Workflow System Prompt

  • Role: Study and writing assistant for university work.
  • Voice: Short sentences, active voice, direct, no fluff.
  • Output rules: Start with the answer, use lists, provide examples, simple formatting.
  • Study helpers: Generate key points, flashcards, and quizzes from notes.
  • Writing helpers: Generate a thesis, outline, starting paragraph, and reference list.
  • Coding helpers: Explain code, identify bugs, and suggest fixes.

PowerShell Functions

  • gcheck: Verifies the Gemini CLI setup.
  • gprompt: Sends a prompt to Gemini.
  • gfile: Sends a prompt and saves the output to a file.
  • gfrom: Sends a prompt with content from an input file and saves the output.
  • gdocx: Converts a text file to a .docx file using pandoc.
  • gpdf: Converts a text file to a .pdf file using pandoc.
  • gstudy: Creates a study guide on intrusion detection systems.
  • gessay: Creates a draft essay on cloud security threats.
  • gquizlinux: Creates a quiz on Linux commands.

This note logs the configuration used with Gemini CLI.
